Item.

Technically, an element of \(\mathbb{Z} \times \mathbb{Z}\) is an ordered pair, and so we should write \(f( {( {m, n} )} )\) for the output of the function \(f\) when the input is the ordered pair \(\left( {m, n} \right)\text{.}\) However, the double parentheses seem unnecessary in this context and there should be no confusion if we write \(f( {m, n} )\) for the output of the function \(f\) when the input is \(\left( {m, n} \right)\text{.}\) So, for example, we simply write

\begin{align*} f( {3, 2} ) \amp = 2 \cdot 3 + 2 = 8,\text{ and }\\ f( { - 4, 5} ) \amp = 2 \cdot \left( { - 4} \right) + 5 = - 3\text{.} \end{align*}
