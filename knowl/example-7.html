Example 2.23.

\begin{equation*} \left( {\exists x \in \mathbb{R}} \right)\left( {x^2 = 5} \right) \end{equation*}

  • There exists a real number \(x\) such that \(x^2 = 5\text{.}\)

  • \(x^2 = 5\) for some real number \(x\text{.}\)

  • There is a real number whose square equals 5.

The second example is usually not used since it is not considered good writing practice to start a sentence with a mathematical symbol. If the universal set is \(\R\text{,}\) then the truth set of the predicate “\(x^2 = 5\)” is \(\left\{ { - \sqrt 5 ,\;\sqrt 5 } \right\}\text{.}\) So these are all true statements.
