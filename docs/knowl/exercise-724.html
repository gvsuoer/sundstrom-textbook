Exercise 11.

Let \(A\) be a nonempty set and let \(\sim\) be an equivalence relation on \(A\text{.}\) Prove each of the following:

(a)

For each \(a, b \in A\text{,}\) \(a\nsim b\) if and only if \(\left[ a \right] \cap \left[ b \right] = \emptyset\text{.}\)

(b)

For each \(a, b \in A\text{,}\) if \(\left[ a \right] \ne \left[ b \right]\text{,}\) then \(\left[ a \right] \cap \left[ b \right] = \emptyset\text{.}\)

(c)

For each \(a, b \in A\text{,}\) if \(\left[ a \right] \cap \left[ b \right] \ne \emptyset\) then \(\left[ a \right] = \left[ b \right]\text{.}\)
