Let \(A = \mathbb{Z} \times \left( {\mathbb{Z} - \left\{ 0 \right\}} \right)\text{.}\) That is, \(A = \left\{ { {\left( {a, b} \right) \in \mathbb{Z} \times \mathbb{Z} } \mid b \ne 0} \right\}\text{.}\) Define the relation \(\approx\) on \(A\) as follows:

For \(\left( {a, b} \right), \left( {c, d} \right) \in A\text{,}\) \(\left( {a, b} \right) \approx \left( {c, d} \right)\) if and only if \(ad = bc\text{.}\)
