A relation was studied in each of the beginning activities for this section. For Beginning Activity 2, the set \(S = \left\{ (x, y) \in \R \times \R \mid 4x^2 + y^2 = 16 \right\}\!\) is a subset of \(\R \times \R\) and, hence, \(S\) is a relation on \(\R\text{.}\) In ExerciseÂ 3 of Beginning Activity 2, we actually determined the domain and range of this relation.

\begin{align*} \text{ dom} (S) \amp = A = \left\{ x \in \R \mid \text{ there exists a } y \in \R \text{ such that } 4x^2 + y^2 = 16 \right\}\\ \text{ range} (S) \amp = B = \left\{ y \in \R \mid \text{ there exists an } x \in \R \text{ such that } 4x^2 + y^2 = 16 \right\} \end{align*}
