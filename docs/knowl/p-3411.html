The intersection of \(\mathbf{\mathcal{C}}\) is defined as the set of all elements that are in all of the sets in \(\mathcal{C}\text{.}\) That is,

\begin{equation*} \bigcap_{X \in \mathscr{C}}^{}X = \left\{x \in U \mid x \in X \text{ for all } X \in \mathscr{C} \right\} \end{equation*}
