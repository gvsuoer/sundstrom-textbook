Activity 44. A Partition Defines an Equivalence Relation.

Let \(A = \left\{ {a, b, c, d, e} \right\}\) and let \(\mathcal{C} = \left\{ {\left\{ {a, b, c} \right\}, \left\{ {d, e} \right\}} \right\}\text{.}\)

(a)

Explain why \(\mathcal{C}\) is a partition of \(A\text{.}\)

(b)

Define a relation \(\sim\) on \(A\) as follows: For \(x, y \in A\text{,}\) \(x \sim y\) if and only if there exists a set \(U\) in \(\mathcal{C}\) such that \(x \in U\) and \(y \in U\text{.}\)

Prove that \(\sim\) is an equivalence relation on the set \(A\text{,}\) and then determine all the equivalence classes for \(\sim\text{.}\) How does the collection of all equivalence classes compare to \(\mathcal{C}\text{?}\)

(c)

What we did for the specific partition in Task 44.b can be done for any partition of a set. So to generalize Task 44.b, we let \(A\) be a nonempty set and let \(\mathcal{C}\) be a partition of \(A\text{.}\) We then define a relation \(\sim\) on \(A\) as follows:

For \(x, y \in A\text{,}\) \(x \sim y\) if and only if there exists a set \(U\) in \(\mathcal{C}\) such that \(x \in U\) and \(y \in U\text{.}\)

Prove that \(\sim\) is an equivalence relation on the set \(A\text{.}\)

(d)

Let \(a \in A\) and let \(U \in \mathcal{C}\) such that \(a \in U\text{.}\) Prove that \([ a ] = U\text{.}\)
