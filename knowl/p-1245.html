In SectionÂ 1.2 and so far in this section, our focus has been on proving statements that involve universal quantifiers. However, another important skill for mathematicians is to be able to recognize when a statement is false and then to be able to prove that it is false. For example, suppose we want to know if the following proposition is true or false.

For each integer \(n\text{,}\) if 5 divides \(\left(n^2 - 1 \right)\text{,}\) then 5 divides \(\left( n - 1 \right)\text{.}\)
