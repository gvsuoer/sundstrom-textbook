Exercise 5.

Let \(\Z_{9} = \left\{ 0, 1, 2, 3, 4, 5, 6, 7, 8 \right\}\text{.}\)

(a)

Define the relation \(\sim\) on \(\Z_{9}\) as follows: For all \(a, b \in \Z_{9}\text{,}\) \(a \sim b\) if and only if \(a^2 \equiv b^2 \pmod {9}\text{.}\) Prove that \(\sim\) is an equivalence relation on \(\Z_{9}\) and determine all of the distinct equivalence classes of this equivalence relation.

Answer.
