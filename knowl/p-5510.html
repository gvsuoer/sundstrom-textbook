Let \(a\text{,}\) \(b\text{,}\) and \(c\) be integers with \(a \ne 0\) and \(b \ne 0\text{,}\) and let \(d = \gcd( a, b )\text{.}\)

  1. If \(d\) does not divide \(c\text{,}\) then the linear Diophantine equation \(ax + by = c\) has no solution.

  2. If \(d\) divides \(c\text{,}\) then the linear Diophantine equation \(ax + by = c\) has infinitely many solutions. In addition, if \(\left( x_0, y_0 \right)\) is a particular solution of this equation, then all the solutions of this equation can be written in the form

    \begin{equation*} x = x_0 + \frac{b}{d} k \text{ and } y = y_0 - \frac{a}{d} k\text{,} \end{equation*}

    for some integer \(k\text{.}\)
