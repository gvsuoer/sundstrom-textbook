(a)

\(A - \left(A \cap B^c \right) = A \cap B\)

(b)

\(\left( A^c \cup B \right)^c \cap A = A - B\)

(c)

\(\left( A \cup B \right) - A = B - A\)

(d)

\(\left( A \cup B \right) - B = A - \left( A \cap B \right)\)
