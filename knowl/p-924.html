Using this notation, the statement “For each real number \(x\text{,}\) \(x^2 > 0\)” could be written in symbolic form as: \(\left( {\forall x \in \mathbb{R}} \right)\left( {x^2 > 0} \right)\text{.}\) The following is an example of a statement involving an existential quantifier.

There exists an integer \(x\) such that \(3x - 2 = 0\text{.}\)
This could be written in symbolic form as

\begin{equation*} \left( {\exists x \in \Z} \right) \left( 3x - 2 = 0 \right)\text{.} \end{equation*}

This statement is false because there are no integers that are solutions of the linear equation \(3x - 2 = 0\text{.}\) Table 2.21 summarizes the facts about the two types of quantifiers.
