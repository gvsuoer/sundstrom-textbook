Subsection Functions Acting on Sets

In our study of functions, we have focused on how a function “maps” individual elements of its domain to the codomain. We also studied the preimage of an individual element in its codomain. For example, if \(f\x \mathbb{R} \to \mathbb{R}\) is defined by \(f ( x ) = x^2\text{,}\) for each \(x \in \mathbb{R}\text{,}\) then

  • \(f ( 2 ) = 4\text{.}\) We say that \(f\) maps 2 to 4 or that 4 is the image of 2 under the function \(f\text{.}\)

  • Since \(f ( x ) = 4\) implies that \(x = 2\) or \(x = -2\text{,}\) we say that the preimages of 4 are 2 and \(-2\) or that the set of preimages of 4 is \(\left\{ -2, 2 \right\}\text{.}\)

For a function \(f\x S \to T\text{,}\) the next step is to consider subsets of \(S\) or \(T\) and what corresponds to them in the other set. We did this in the beginning activities. We will give some definitions and then revisit the examples in the beginning activities in light of these definitions. We will first consider the situation where \(A\) is a subset of \(S\) and consider the set of outputs whose inputs are from \(A\text{.}\) This will be a subset of \(T\text{.}\)

Definition 6.6.1.

Let \(f\x S \to T\text{.}\) If \(A \subseteq S\text{,}\) then the image of \(\boldsymbol{A}\) under \(\boldsymbol{f}\) is the set \(f ( A )\text{,}\) where

\begin{equation*} f ( A ) = \left\{f ( x ) \mid x \in A \right\}\!\text{.} \end{equation*}

If there is no confusion as to which function is being used, we call \(f ( A )\) the image of \(\boldsymbol{A}\).

We now consider the situation in which \(C\) is a subset of \(T\) and consider the subset of \(A\) consisting of all elements of \(T\) whose outputs are in \(C\text{.}\)

Definition 6.6.2.

Let \(f\x S \to T\text{.}\) If \(C \subseteq T\text{,}\) then the preimage of \(\boldsymbol{C}\) under \(\boldsymbol{f}\) is the set \(f^{-1} ( C )\text{,}\) where

\begin{equation*} f^{-1} ( C ) = \left\{x \in S \mid f ( x ) \in C \right\}\text{.} \end{equation*}

If there is no confusion as to which function is being used, we call \(f^{-1} ( C )\) the preimage of \(\boldsymbol{C}\). The preimage of the set \(C\) under \(f\) is also called the inverse image of \(\boldsymbol{C}\) under \(\boldsymbol{f}\).

Notice that the set \(f^{-1} ( C )\) is defined whether or not \(f^{-1}\) is a function.

Subsection Set Operations and Functions Acting on Sets

We will now consider the following situation: Let \(S\) and \(T\) be sets and let \(f\) be a function from \(S\) to \(T\text{.}\) Also, let \(A\) and \(B\) be subsets of \(S\) and let \(C\) and \(D\) be subsets of \(T\text{.}\) In the remainder of this section, we will consider the following situations and answer the questions posed in each case.

  • The set \(A \cap B\) is a subset of \(S\) and so \(f ( A \cap B )\) is a subset of \(T\text{.}\) In addition, \(f ( A )\) and \(f ( B )\) are subsets of \(T\text{.}\) Hence, \(f ( A ) \cap f ( B )\) is a subset of \(T\text{.}\)

    Is there any relationship between \(f ( A \cap B )\) and \(f ( A ) \cap f ( B )\text{?}\)

  • The set \(A \cup B\) is a subset of \(S\) and so \(f ( A \cup B )\) is a subset of \(T\text{.}\) In addition, \(f ( A )\) and \(f ( B )\) are subsets of \(T\text{.}\) Hence, \(f ( A ) \cup f ( B )\) is a subset of \(T\text{.}\)

    Is there any relationship between \(f ( A \cup B )\) and \(f ( A ) \cup f ( B )\text{?}\)

  • The set \(C \cap D\) is a subset of \(T\) and so \(f^{-1} ( C \cap D )\) is a subset of \(S\text{.}\) In addition, \(f^{-1} ( C )\) and \(f^{-1} ( D )\) are subsets of \(S\text{.}\) Hence, \(f^{-1} ( C ) \cap f^{-1} ( D )\) is a subset of \(S\text{.}\)

    Is there any relationship between the sets \(f^{-1} ( C \cap D )\) and \(f^{-1} ( C ) \cap f^{-1} ( D )\text{?}\)

  • The set \(C \cup D\) is a subset of \(T\) and so \(f^{-1} ( C \cup D )\) is a subset of \(S\text{.}\) In addition, \(f^{-1} ( C )\) and \(f^{-1} ( D )\) are subsets of \(S\text{.}\) Hence, \(f^{-1} ( C ) \cup f^{-1} ( D )\) is a subset of \(S\text{.}\)

    Is there any relationship between the sets \(f^{-1} ( C \cup D )\) and \(f^{-1} ( C ) \cup f^{-1} ( D )\text{?}\)

These and other questions will be explored in the next progress check.

The examples in Progress Check 6.6.5 and Progress Check 6.6.6 were meant to illustrate general results about how functions act on sets. In particular, we investigated how the action of a function on sets interacts with the set operations of intersection and union. We will now state the theorems that these examples were meant to illustrate. Some of the proofs will be left as exercises.

Theorem 6.6.8.

Let \(f\x S \to T\) be a function and let \(A\) and \(B\) be subsets of \(S\text{.}\) Then

  1. \(f ( A \cap B ) \subseteq f ( A ) \cap f ( B )\)

  2. \(f ( A \cup B ) = f ( A ) \cup f ( B )\)

Proof.

We will prove Item 1. The proof of Item 2 is Exercise 5.

Assume that \(f\x S \to T\) is a function and let \(A\) and \(B\) be subsets of \(S\text{.}\) We will prove that \(f ( A \cap B ) \subseteq f ( A ) \cap f ( B )\) by proving that for all \(y \in T\text{,}\) if \(y \in f ( A \cap B )\text{,}\) then \(y \in f ( A ) \cap f ( B )\text{.}\)

We assume that \(y \in f ( A \cap B )\text{.}\) This means that there exists an \(x \in A \cap B\) such that \(f ( x ) = y\text{.}\) Since \(x \in A \cap B\text{,}\) we conclude that \(x \in A\) and \(x \in B\text{.}\)

  • Since \(x \in A\) and \(f ( x ) = y\text{,}\) we conclude that \(y \in f ( A )\text{.}\)

  • Since \(x \in B\) and \(f ( x ) = y\text{,}\) we conclude that \(y \in f ( B )\text{.}\)

Since \(y \in f ( A )\) and \(y \in f ( B )\text{,}\) \(y \in f ( A ) \cap f ( B )\text{.}\) This proves that if \(y \in f ( A \cap B )\text{,}\) then \(y \in f ( A ) \cap f ( B )\text{.}\) Hence \(f ( A \cap B ) \subseteq f ( A ) \cap f ( B )\text{.}\)

Theorem 6.6.9.

Let \(f\x S \to T\) be a function and let \(C\) and \(D\) be subsets of \(T\text{.}\) Then

  1. \(f^{-1} ( C \cap D ) = f^{-1} ( C ) \cap f^{-1} ( D )\)

  2. \(f^{-1} ( C \cup D ) = f^{-1} ( C ) \cup f^{-1} ( D )\)

Proof.

We will prove Item 2. The proof of Item 1 is Exercise 6.

Assume that \(f\x S \to T\) is a function and that \(C\) and \(D\) are subsets of \(T\text{.}\) We will prove that \(f^{-1} ( C \cup D ) = f^{-1} ( C ) \cup f^{-1} ( D )\) by proving that each set is a subset of the other.

We start by letting \(x\) be an element of \(f^{-1} ( C \cup D )\text{.}\) This means that \(f ( x )\) is an element of \(C \cup D\text{.}\) Hence,

\begin{equation*} f ( x ) \in C \text{ or } f ( x ) \in D\text{.} \end{equation*}

In the case where \(f ( x ) \in C\text{,}\) we conclude that \(x \in f^{-1} ( C )\text{,}\) and hence that \(x \in f^{-1} ( C ) \cup f^{-1} ( D )\text{.}\) In the case where \(f ( x ) \in D\text{,}\) we see that \(x \in f^{-1} ( D )\text{,}\) and hence that \(x \in f^{-1} ( C ) \cup f^{-1} ( D )\text{.}\) So in both cases, \(x \in f^{-1} ( C ) \cup f^{-1} ( D )\text{,}\) and we have proved that \(f^{-1} ( C \cup D ) \subseteq f^{-1} ( C ) \cup f^{-1} ( D )\text{.}\)

We now let \(t \in f^{-1} ( C ) \cup f^{-1} ( D )\text{.}\) This means that

\begin{equation*} t \in f^{-1} ( C ) \text{ or } t \in f^{-1} ( D )\text{.} \end{equation*}

  • In the case where \(t \in f^{-1} ( C )\text{,}\) we conclude that \(f ( t ) \in C\) and hence that \(f ( t ) \in C \cup D\text{.}\) This means that \(t \in f^{-1} ( C \cup D )\text{.}\)

  • Similarly, when \(t \in f^{-1} ( D )\text{,}\) it follows that \(f ( t ) \in D\) and hence that \(f ( t ) \in C \cup D\text{.}\) This means that \(t \in f^{-1} ( C \cup D )\text{.}\)

These two cases prove that if \(t \in f^{-1} ( C ) \cup f^{-1} ( D )\text{,}\) then \(t \in f^{-1} ( C \cup D )\text{.}\) Therefore, \(f^{-1} ( C ) \cup f^{-1} ( D ) \subseteq f^{-1} ( C \cup D )\text{.}\)

Since we have now proved that each of the two sets is a subset of the other set, we can conclude that \(f^{-1} ( C \cup D ) = f^{-1} ( C ) \cup f^{-1} ( D )\text{.}\)

Theorem 6.6.10.

Let \(f\x S \to T\) be a function and let \(A\) be a subset of \(S\) and let \(C\) be a subset of \(T\text{.}\) Then

  1. \(\displaystyle A \subseteq f^{-1} \!\left( f ( A ) \right)\)

  2. \(\displaystyle f \!\left( f^{-1} ( C ) \right) \subseteq C\)

Proof.

We will prove Item 1. The proof of Item 2 is Exercise 7.

To prove Item 1, we will prove that for all \(a \in S\text{,}\) if \(a \in A\text{,}\) then

\(a \in f^{-1} \!\left( f ( A ) \right)\text{.}\) So let \(a \in A\text{.}\) Then, by definition, \(f ( a ) \in f ( A )\text{.}\) We know that \(f ( A ) \subseteq T\text{,}\) and so \(f^{-1} \!\left( f ( A ) \right) \subseteq S\text{.}\) Notice that

\begin{equation*} f^{-1} \!\left( f ( A ) \right) = \left\{ x \in S \mid f ( x ) \in f ( A ) \right\}\text{.} \end{equation*}

Since \(f ( a ) \in f ( A )\text{,}\) we use this to conclude that \(a \in f^{-1}\!\left ( f ( A ) \right)\text{.}\) This proves that if \(a \in A\text{,}\) then \(a \in f^{-1} \!\left( f ( A ) \right)\text{,}\) and hence that \(A \subseteq f^{-1} \!\left( f ( A ) \right)\text{.}\)
