In the second to the last example, the quantifier is not stated explicitly. Care must be taken when reading this because it really does say the same thing as the previous examples. The last example illustrates the fact that conditional statements often contain a “hidden” universal quantifier. If the universal set is \(\R\text{,}\) then the truth set of the open sentence \(x^2 > 0\) is the set of all nonzero real numbers. That is, the truth set is

\begin{equation*} \left\{ {x \in \mathbb{R}} \mid x \ne 0 \right\}\text{.} \end{equation*}

So the preceding statements are false. For the conditional statement, the example using \(x = 0\) produces a true hypothesis and a false conclusion. This is a counterexample that shows that the statement with a universal quantifier is false.
