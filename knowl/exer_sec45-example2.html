Exercise 2.

For each natural number \(n\text{,}\) let \(A_n = \left\{ k \in \N \left| \hspace{3pt} k \geq n \right. \right\}\text{.}\) Assuming the universal set is \(\N\text{,}\) use the roster method or set builder notation to specify each of the following sets:

(a)

\(\bigcap\limits_{j=1}^{5}A_j\)

Answer.
(b)

\(\left(\: \bigcap\limits_{j=1}^{5}A_j \right)^c\)

(c)

\(\bigcap\limits_{j=1}^{5}A_j^c\)

Answer.
(d)

\(\bigcup\limits_{j=1}^{5}A_j^c\)

Answer.
(e)

\(\bigcup\limits_{j=1}^{5}A_j\)

(f)

\(\left(\: \bigcup\limits_{j=1}^{5}A_j \right)^c\)

Answer.
(g)

\(\bigcap\limits_{j \in \N}^{}A_j\)

(h)

\(\bigcup\limits_{j \in \N}^{}A_j\)
