Activity 34. Creating Functions with Finite Domains.

Let \(A = \left\{ {a,b,c,d} \right\}\text{,}\) \(B = \left\{ {a,b,c} \right\}\text{,}\) and \(C = \left\{ {s,t,u,v} \right\}\text{.}\) In each of the following exercises, draw an arrow diagram to represent your function when it is appropriate.

(a)

Create a function \(f\x A \to C\) whose range is the set \(C\) or explain why it is not possible to construct such a function.

(b)

Create a function \(f\x A \to C\) whose range is the set \(\left\{ {u, v} \right\}\) or explain why it is not possible to construct such a function.

(c)

Create a function \(f\x B \to C\) whose range is the set \(C\) or explain why it is not possible to construct such a function.

(d)

Create a function \(f\x A \to C\) whose range is the set \(\left\{ u \right\}\) or explain why it is not possible to construct such a function.

(e)

If possible, create a function \(f\x A \to C\) that satisfies the following condition:

For all \(x, y \in A\text{,}\) if \(x \ne y\text{,}\) then \(f( x ) \ne f( y )\text{.}\)
If it is not possible to create such a function, explain why.

(f)

If possible, create a function \(f\x A \to \left\{ {s, t, u} \right\}\) that satisfies the following condition:

For all \(x, y \in A\text{,}\) if \(x \ne y\text{,}\) then \(f( x ) \ne f( y )\text{.}\)
If it is not possible to create such a function, explain why.
