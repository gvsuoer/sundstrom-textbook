Item 3.

\(\left( {\exists x \in \mathbb{Z}} \right)\left( {\forall y \in \mathbb{Z}} \right)\left( {x + y = 0} \right)\)

We could read this as, “There exists an integer \(x\) such that for each integer \(y\text{,}\) \(x + y = 0\text{.}\)” This is a false statement since there is no integer whose sum with each integer is zero.
