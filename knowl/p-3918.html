Also, the definition of a function does not require that the range of the function must equal the codomain. The range is always a subset of the codomain, but these two sets are not required to be equal. That is, if \(g\x A \to B\text{,}\) then it is possible to have a \(y \in B\) such that \(g( x ) \ne y\) for all \(x \in A\text{.}\) The arrow diagram for the function \(g\) in FigureÂ 6.15 illustrates such a function.
