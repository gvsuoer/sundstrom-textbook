The conditional statement \(P \to Q\) means that \(Q\) is true whenever \(P\) is true. It says nothing about the truth value of \(Q\) when \(P\) is false. Using this as a guide, we define the conditional statement \(P \to Q\) to be false only when \(P\) is true and \(Q\) is false, that is, only when the hypothesis is true and the conclusion is false. In all other cases, \(P \to Q\) is true.
