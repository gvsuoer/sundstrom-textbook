Item 4.

\(\left( {\exists x \in \mathbb{Z}} \right)\left( {\exists y \in \mathbb{Z}} \right)\left( {x + y = 0} \right)\)

We could read this as, “There exist integers \(x\) and \(y\) such that \(x + y = 0\text{.}\)” This is a true statement. For example, \(2 + \left( { - 2} \right) = 0\text{.}\)
