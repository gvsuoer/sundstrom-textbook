Now let \(\Lambda = \R\text{.}\) For each \(x \in \R\text{,}\) define \(B_x = \left\{ 0, x^2, x^4 \right\}\text{.}\) Is the following statement true for the indexed family of sets \(\mathscr{B} = \left\{ B_x \mid x \in \R \right\}\text{?}\)

For all \(x, y \in \R\text{,}\) if \(x \ne y\text{,}\) then \(B_x \ne B_y\text{.}\)
