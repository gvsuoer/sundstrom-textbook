We need to use set builder notation for the set \(\Q\) of all rational numbers, which consists of quotients of integers.

\begin{equation*} \Q = \left\{ \left. \frac{m}{n} \right| m, n \in \Z \text{ and } n \ne 0 \right\} \end{equation*}

Since any integer \(n\) can be written as \(n = \dfrac{n}{1}\text{,}\) we see that \(\Z \subseteq \Q\text{.}\)
