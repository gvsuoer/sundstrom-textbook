Let \(A = \left\{ x \in \mathbb{R} \mid x\left( {x + 2} \right)^2 \left( x - \frac{3}{2} \right) = 0 \right\}\text{.}\) Which of the following sets are equal to the set \(A\) and which are subsets of \(A\text{?}\)

(a)

\(\left\{ { - 2, 0,3} \right\}\)

(b)

\(\left\{ {\dfrac{3}{2}, - 2,0} \right\}\)

Answer.
(c)

\(\left\{ { - 2, - 2,0,\dfrac{3}{2}} \right\}\)

Answer.
(d)

\(\left\{ { - 2, \dfrac{3}{2}} \right\}\)
