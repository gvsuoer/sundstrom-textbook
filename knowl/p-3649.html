When we have a function \(f \colon A \to B\text{,}\) we often write \(y = f(x)\text{.}\) In this case, we consider \(x\) to be an unspecified object that can be chosen from the set \(A\text{,}\) and we would say that \(x\) is the independent variable of the function \(f\) and \(y\) is the dependent variable of the function \(f\text{.}\)
