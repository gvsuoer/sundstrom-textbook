Now, let \(a, b \in \mathbb{R}\) and assume that \(a \sim b\text{.}\) We will prove that \(b \sim a\text{.}\) Since \(a \sim b\text{,}\) there exists an integer \(k\) such that

\begin{equation*} a - b = 2k\pi\text{.} \end{equation*}

By multiplying both sides of this equation by \(-1\text{,}\) we obtain

\begin{align*} ( { - 1} )( {a - b} ) \amp = ( { - 1} )( {2k\pi } )\\ b = a \amp = 2( { - k} )\pi\text{.} \end{align*}

Since \(- k \in \mathbb{Z}\text{,}\) the last equation proves that \(b \sim a\text{.}\) Hence, we have proven that if \(a \sim b\text{,}\) then \(b \sim a\) and, therefore, the relation \(\sim\) is symmetric.
