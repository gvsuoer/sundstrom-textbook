Let \(S = \left\{1, 2, 3, 4 \right\}\text{.}\)

(a)

Define \(F\x S \to \mathbb{N}\) by \(F ( x ) = x^2\) for each \(x \in S\text{.}\) What is the range of the function \(F\) and what is \(F ( S )\text{?}\) How do these two sets compare?

Answer.
(b)

Now let \(A\) and \(B\) be sets and let \(f\x A \to B\) be an arbitrary function from \(A\) to \(B\text{.}\)

Explain why \(f ( A ) = \text{ range} ( f )\text{.}\)

(c)

Define a function \(g\x A \to f ( A )\) by \(g ( x ) = f ( x )\) for all \(x\) in \(A\text{.}\) Prove that the function \(g\) is a surjection.
