Definition.

Let \(\Lambda\) be a nonempty set and suppose that for each \(\alpha \in \Lambda\text{,}\) there is a corresponding set \(A_\alpha\text{.}\) The family of sets \(\left\{ A_\alpha \mid \alpha \in \Lambda \right\}\) is called an indexed family of sets indexed by \(\Lambda\text{.}\) Each \(\alpha \in \Lambda\) is called an index and \(\Lambda\) is called an indexing set.
