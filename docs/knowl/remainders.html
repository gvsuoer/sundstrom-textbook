Theorem 3.37 and Corollary 3.38 state that an integer is congruent (mod \(n\)) to its remainder when it is divided by \(n\text{.}\) (Recall that we always mean the remainder guaranteed by the Division Algorithm, which is the least nonnegative remainder.) Since this remainder is unique and since the only possible remainders for division by \(n\) are \(0, 1, 2, \ldots , n - 1\text{,}\) we then know that each integer is congruent, modulo \(n\text{,}\) to precisely one of the integers \(0,1,2, \ldots ,n - 1\text{.}\) So for each natural number \(n\text{,}\) we will define a new set \(R_n\) as the set of remainders upon division by \(n\text{.}\) So

\begin{equation*} R_n = \{ 0, 1, 2, \ldots , n - 1 \}\text{.} \end{equation*}

For example, \(R_4 = \{0, 1, 2, 3 \}\) and \(R_6 = \{ 0, 1, 2, 3, 4, 5 \}\text{.}\) We will now explore a method to define a function from \(R_6\) to \(R_6\text{.}\)
