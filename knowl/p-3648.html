Let \(f \colon A \to B\text{.}\) (This is read, “Let \(f\) be a function from \(A\) to \(B\text{.}\)” ) The set \(A\) is called the domain of the function \(f\text{,}\) and we write \(A = \text{ dom} ( f )\text{.}\) The set \(B\) is called the codomain of the function \(f\text{,}\) and we write \(B = \text{ codom} ( f )\text{.}\) If \(a \in A\text{,}\) then the element of \(B\) that is associated with \(a\) is denoted by \(f( a )\) and is called the image of \(\boldsymbol{a}\) under \(\boldsymbol{f}\). If \(f( a ) = b\text{,}\) with \(b \in B\text{,}\) then \(a\) is called a preimage of \(\boldsymbol{b}\) for \(\boldsymbol{f}\).
