We now assume that \(y \in [ b ]\text{.}\) This means that \(y \sim b\text{,}\) and hence by the symmetric property, that \(b \sim y\text{.}\) Again, we are assuming that \(a \sim b\text{.}\) So we have \(a \sim b\) and \(b \sim y\text{.}\) We use the transitive property to conclude that \(a \sim y\) and then, using the symmetric property, we conclude that \(y \sim a\text{.}\) This proves that \(y \in [ a ]\) and, hence, that \([ b ] \subseteq [ a ]\text{.}\) This means that we can conclude that if \(a \sim b\text{,}\) then \([ a ] = [ b ]\text{.}\)
