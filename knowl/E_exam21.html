Example 2.15. Open Sentences.

If the universal set is \(\mathbb{R}\text{,}\) then the sentence “\(x^2 - 3x - 10 = 0\)” is an open sentence involving the one variable \(x\text{.}\)

If we substitute \(x = 2\text{,}\) we obtain the false statement “\(2^2 - 3 \cdot 2 - 10 = 0\text{.}\)”
If we substitute \(x = 5\text{,}\) we obtain the true statement “\(5^2 - 3 \cdot 5 - 10 = 0\text{.}\)”
In this example, we can let \(P( x )\) be the predicate “\(x^2 - 3x - 10 = 0\)” and then say that \(P( 2 )\) is false and \(P( 5 )\) is true.

Using similar notation, we can let \(Q( {x, y} )\) be the predicate “\(x + 2y = 7\text{.}\)” This predicate involves two variables. Then,

\(Q( {1, 1})\) is false since “\(1 + 2 \cdot 1 = 7\)” is false; and
\(Q( {3, 2} )\) is true since “\(3 + 2 \cdot 2 = 7\)” is true.
