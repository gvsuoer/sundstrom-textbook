Let \(\Z^*\) be the set of all nonzero integers.

(a)

Use a counterexample to explain why the following statement is false:

For each \(x \in \Z^*\text{,}\) there exists a \(y \in \Z^*\) such that \(xy = 1\text{.}\)

(b)

Write the statement in Task a in symbolic form using appropriate symbols for quantifiers.

(c)

Write the negation of the statement in Task b in symbolic form using appropriate symbols for quantifiers.

(d)

Write the negation from Task c in English without using the symbols for quantifiers.
