Activity 35. Integration as a Function.

In calculus, we learned that if \(f\) is real function that is continuous on the closed interval \(\left[ {a, b} \right]\text{,}\) then the definite integral \(\int_a^b {f( x ) \, dx}\) is a real number. In fact, one form of the Fundamental Theorem of Calculus states that

\begin{equation*} \int_a^b {f( x ) \, dx} = F( b ) - F( a )\text{,} \end{equation*}

where \(F\) is any antiderivative of \(f\text{,}\) that is, where \(F \,' = f\text{.}\)

(a)

Let \(\left[ {a, b} \right]\) be a closed interval of real numbers and let \(C {\left[ {a, b} \right]}\) be the set of all real functions that are continuous on \(\left[ {a, b} \right]\text{.}\) That is,

\begin{equation*} C{\left[ {a, b} \right]} = \left\{ {f\x \left[ {a, b} \right] \to \mathbb{R} \mid f\text{ is continuous on } \left[ {a, b} \right]} \right\}\text{.} \end{equation*}
(i)

Explain how the definite integral \(\int_a^b {f( x ) \, dx}\) can be used to define a function \(I\) from \(C{\left[ {a, b} \right]}\) to \(\mathbb{R}\text{.}\)

(ii)

Let \(\left[ {a, b} \right] = \left[ {0, 2} \right]\text{.}\) Calculate \(I( f )\text{,}\) where \(f( x ) = x^2 + 1\text{.}\)

(iii)

Let \(\left[ {a, b} \right] = \left[ {0, 2} \right]\text{.}\) Calculate \(I( g )\text{,}\) where \(g( x ) = \sin ( {\pi x} )\text{.}\)

In calculus, we also learned how to determine the indefinite integral \(\int {f( x )} \;dx\) of a continuous function \(f\text{.}\)

(b)

Let \(f( x ) = x^2 + 1\) and \(g(x) = \cos( 2x )\text{.}\) Determine \(\int {f( x )} \;dx\) and \(\int {g( x )} \;dx\text{.}\)

(c)

Let \(f\) be a continuous function on the closed interval \(\left[ {0, 1} \right]\) and let \(T\) be the set of all real functions. Can the process of determining the indefinite integral of a continuous function be used to define a function from \(C{\left[ {0, 1} \right]}\) to \(T\text{?}\) Explain.

(d)

Another form of the Fundamental Theorem of Calculus states that if \(f\) is continuous on the interval \(\left[ {a, b} \right]\) and if

\begin{equation*} g( x ) = \int_a^x {f( t ) \, dt} \end{equation*}

for each \(x\) in \(\left[ {a, b} \right]\text{,}\) then \(g'\left( x \right) = f\left( x \right)\text{.}\) That is, \(g\) is an antiderivative of \(f\text{.}\) Explain how this theorem can be used to define a function from \(C{\left[ {a,\;b} \right]}\) to \(T\text{,}\) where the output of the function is an antiderivative of the input. (Recall that \(T\) is the set of all real functions.)
