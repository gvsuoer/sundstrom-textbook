Progress Check 6.1.3. Images and Preimages.

Let \(f\x \R \to \R\) be defined by \(f(x) = x^2 - 5x\) for all \(x \in \R\text{,}\) and let \(g\x \Z \to \Z\) be defined by \(g(m) = m^2 - 5m\) for all \(m \in \Z\text{.}\)

(a)

Determine \(f ( -3 )\) and \(f \left( \sqrt 8 \right)\text{.}\)

(b)

Determine \(g ( 2 )\) and \(g ( -2 )\text{.}\)

(c)

Determine the set of all preimages of 6 for the function \(f\text{.}\)

(d)

Determine the set of all preimages of 6 for the function \(g\text{.}\)

(e)

Determine the set of all preimages of 2 for the function \(f\text{.}\)

(f)

Determine the set of all preimages of 2 for the function \(g\text{.}\)
