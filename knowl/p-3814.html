Let \(A = \left\{ a_1, a_2, \ldots, a_n \right\}\) be a finite set whose elements are the distinct real numbers \(a_1, a_2, \ldots, a_n\text{.}\) We define the average of the set \(A\) to be the real number \(\bar{A}\text{,}\) where

\begin{equation*} \bar{A} = \frac{a_1 + a_2 + \cdots + a_n}{n}\text{.} \end{equation*}
