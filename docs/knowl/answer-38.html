Answer 5.b.1.

We assume that \(m\) is an odd integer and will prove that \(3m^2 + 7m + 12\) is an even integer. Since \(m\) is odd, there exists an integer \(k\) such that \(m = 2k +1\text{.}\) Hence,

\begin{align*} 3m^2 + 7m + 12 \amp = 3 \left( 2k + 1 \right)^2 + 7 \left( 2k + 1 \right) + 12\\ \amp = 12k^2 + 26k + 22\\ \amp = 2 \left( 6k^2 + 13k + 11 \right) \end{align*}

By the closure properties of the integers, \(\left( 6k^2 + 13k + 11 \right)\) is an integer. Hence, this proves that if \(m\) is odd, then \(3m^2 + 7m + 12\) is an even integer.
