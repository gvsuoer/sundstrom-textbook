Example 1.17. Truth Sets.

Assume the universal set is \(\R\) and \(P( x )\) is “\(x^2 \lt 4\text{.}\)” We can describe the truth set of \(P(x)\) as the set of all real numbers whose square is less than 4. We can also use set builder notation to write the truth set of \(P( x )\) as

\begin{equation*} \left\{ {x \in \R} \mid x^2 \lt 4 \right\}\text{.} \end{equation*}

However, if we solve the inequality \(x^2 \lt 4\text{,}\) we obtain \(- 2 \lt x \lt 2\text{.}\) So we could also write the truth set as

\begin{equation*} \left\{ {x \in \R} \mid - 2 \lt x \lt 2 \right\}\text{.} \end{equation*}

We could read this as the set of all real numbers that are greater than \(-2\) and less than 2. We can also write

\begin{equation*} \left\{ {x \in \R} \mid x^2 \lt 4 \right\} = \left\{ {x \in \mathbb{R}} \mid - 2 \lt x \lt 2 \right\}\text{.} \end{equation*}
