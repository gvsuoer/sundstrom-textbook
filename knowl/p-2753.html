If the two sets \(A\) and \(B\) are equal, then it must be true that every element of \(A\) is an element of \(B\text{,}\) that is, \(A \subseteq B\text{,}\) and it must be true that every element of \(B\) is an element of \(A\text{,}\) that is, \(B \subseteq A\text{.}\) Conversely, if \(A \subseteq B\) and \(B \subseteq A\text{,}\) then \(A\) and \(B\) must have precisely the same elements.
