We have already seen that the elements of a set may themselves be sets. For example, the power set of a set \(T\text{,}\) \(\mathcal{P}( T )\text{,}\) is the set of all subsets of \(T\text{.}\) The phrase, “a set of sets” sounds confusing, and so we often use the terms collection and family when we wish to emphasize that the elements of a given set are themselves sets. We would then say that the power set of \(T\) is the family (or collection) of sets that are subsets of \(T\text{.}\)
