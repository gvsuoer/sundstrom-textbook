If \(x \mathrel{R} y\) and \(y \mathrel{R} z\text{,}\)then \(x\) and \(y\) have the same number of digits and \(y\) and \(z\) have the same number of digits. Hence, \(x\) and \(z\) have the same number of digits, and so \(x \mathrel{R} z\text{.}\) Therefore, \(\mathrel{R}\) is transitive.
