In calculus, it can be shown that

\begin{align*} \int_{}^{} \sin^{\,2} x \, dx \amp = \frac{x}{2} - \frac{1}{2} \sin x \cos x + c \text{ and }\\ \int_{}{} \cos^{\,2} x \, dx \amp = \frac{x}{2} + \frac{1}{2} \sin x \cos x + c\text{.} \end{align*}

Using integration by parts, it is also possible to prove that for each natural number \(n\text{,}\)

\begin{align*} \int_{}{} \sin^{\,n} x \, dx \amp = -\frac{1}{n} \sin^{n-1}x \cos x + \frac{n-1}{n} \int{}{} \sin^{\,n-2}x \, dx \text{ and }\\ \int_{}{} \cos^{\,n} x \, dx \amp = \frac{1}{n} \cos^{n-1}x \sin x + \frac{n-1}{n} \int{}{} \cos^{\,n-2}x \, dx\text{.} \end{align*}
