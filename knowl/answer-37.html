Answer 4.a.1.

We assume that \(m\) is an even integer and will prove that \(5m + 7\) is an odd integer. Since \(m\) is an even integer, there exists an integer \(k\) such that \(m = 2k\text{.}\) Using substitution and algebra, we see that

\begin{align*} 5m + 7 \amp = 5(2k) + 7\\ \amp = 10k + 6 + 1\\ \amp = 2(5k + 3) + 1 \end{align*}

By the closure properties of the integers, we conclude that \(5k + 3\) is an integer, and so the last equation proves that \(5m + 7\) is an odd integer.
