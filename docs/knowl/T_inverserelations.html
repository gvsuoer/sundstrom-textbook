Theorem 7.10.

Let \(R\) be a relation from the set \(A\) to the set \(B\text{.}\) Then

  • The domain of \(R^{ - 1}\) is the range of \(R\text{.}\) That is, \(\text{ dom} \!\left( {R^{ - 1} } \right) = \text{ range} ( R )\text{.}\)

  • The range of \(R^{ - 1}\) is the domain of \(R\text{.}\) That is, \(\text{ range} \!\left( {R^{ - 1} } \right) = \text{ dom} ( R )\text{.}\)

  • The inverse of \(R^{ - 1}\) is \(R\text{.}\) That is, \(\left( {R^{ - 1} } \right)^{ - 1} = R\text{.}\)
