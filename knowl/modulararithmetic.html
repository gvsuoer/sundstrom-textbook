Definition.

Let \(n \in \mathbb{N}\text{.}\) Addition and multiplication in \(\mathbb{Z}_n\) are defined as follows: For \([ a ], [ c ] \in \mathbb{Z}_n\text{,}\)

\begin{equation*} [ a ] \oplus [ c ] = [ {a + c} ] \text{ and } [ a ] \odot [ c ] = [ {ac} ]\text{.} \end{equation*}
