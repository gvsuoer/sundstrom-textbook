Activity 45. Equivalence Relations on a Set of Matrices.

The following exercises require a knowledge of elementary linear algebra. We let \(\mathcal{M}_{n, n}\left( \R \right)\) be the set of all \(n\) by \(n\) matrices with real number entries.

(a)

Define a relation \(\sim\) on \(\mathcal{M}_{n, n}( \R )\) as follows: For all \(A, B \in \mathcal{M}_{n, n}( \R )\text{,}\) \(A \sim B\) if and only if there exists an invertible matrix \(P\) in \(\mathcal{M}_{n, n}( \R )\) such that \(B = PAP^{-1}\text{.}\) Is \(\sim\) an equivalence relation on \(\mathcal{M}_{n, n}( \R )\text{?}\) Justify your conclusion.

(b)

Define a relation \(R\) on \(\mathcal{M}_{n, n} (\R )\) as follows: For all \(A, B \in \mathcal{M}_{n, n}( \R )\text{,}\) \(A \mathrel{R} B\) if and only if \(\text{ det } (A) = \text{ det } (B)\text{.}\) Is \(R\) an equivalence relation on \(\mathcal{M}_{n, n}( \R )\text{?}\) Justify your conclusion.

(c)

Let \(\sim\) be an equivalence relation on \(\R\text{.}\) Define a relation \(\approx\) on \(\mathcal{M}_{n, n}( \R )\) as follows: For all \(A, B \in \mathcal{M}_{n, n}( \R )\text{,}\) \(A \approx B\) if and only if \(\text{ det } (A) \sim \text{ det } (B)\text{.}\) Is \(\approx\) an equivalence relation on \(\mathcal{M}_{n, n}( \R )\text{?}\) Justify your conclusion.
