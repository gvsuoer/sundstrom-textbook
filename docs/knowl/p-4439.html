We start by letting \(x\) be an element of \(f^{-1} ( C \cup D )\text{.}\) This means that \(f ( x )\) is an element of \(C \cup D\text{.}\) Hence,

\begin{equation*} f ( x ) \in C \text{ or } f ( x ) \in D\text{.} \end{equation*}

In the case where \(f ( x ) \in C\text{,}\) we conclude that \(x \in f^{-1} ( C )\text{,}\) and hence that \(x \in f^{-1} ( C ) \cup f^{-1} ( D )\text{.}\) In the case where \(f ( x ) \in D\text{,}\) we see that \(x \in f^{-1} ( D )\text{,}\) and hence that \(x \in f^{-1} ( C ) \cup f^{-1} ( D )\text{.}\) So in both cases, \(x \in f^{-1} ( C ) \cup f^{-1} ( D )\text{,}\) and we have proved that \(f^{-1} ( C \cup D ) \subseteq f^{-1} ( C ) \cup f^{-1} ( D )\text{.}\)
