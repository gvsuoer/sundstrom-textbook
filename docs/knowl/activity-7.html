Activity 7. Closure Explorations.

In SectionÂ 1.1, we studied some of the closure properties of the standard number systems. (See Closure Properties of Number Systems.) We can extend this idea to other sets of numbers. So we say that:

  • A set \(A\) of numbers is closed under addition provided that whenever \(x\) and \(y\) are in the set \(A\text{,}\) \(x + y\) is in the set \(A\text{.}\)

  • A set \(A\) of numbers is closed under multiplication provided that whenever \(x\) and \(y\) are are in the set \(A\text{,}\) \(x \cdot y\) is in the set \(A\text{.}\)

  • A set \(A\) of numbers is closed under subtraction provided that whenever \(x\) and \(y\) are are in the set \(A\text{,}\) \(x - y\) is in the set \(A\text{.}\)

For each of the following sets, make a conjecture about whether or not it is closed under addition and whether or not it is closed under multiplication. In some cases, you may be able to find a counterexample that will prove the set is not closed under one of these operations.
