Item 3.

Although one example can be used to prove that a conditional statement is false, in most cases, we cannot use examples to prove that a conditional statement is true. For example, in Progress Check 1.5, we substituted values for \(x\) for the conditional statement “If \(x\) is a positive real number, then \(x^2 + 8x\) is a positive real number.” For every positive real number used for \(x\text{,}\) we saw that \(x^2 + 8x\) was positive. However, this does not prove the conditional statement to be true because it is impossible to substitute every positive real number for \(x\text{.}\) So, although we may believe this statement is true, to be able to conclude it is true, we need to write a mathematical proof. Methods of proof will be discussed in Section 1.2 and Chapter 3.
