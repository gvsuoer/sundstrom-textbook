Item 2.13:1.

If \(A\) is a set and \(y\) is one of the objects in the set \(A\text{,}\) we write \(y \in A\) and read this as “\(y\) is an element of \(A\)” or “\(y\) is a member of \(A\text{.}\)” For example, if \(B\) is the set of all integers greater than 4, then we could write \(5 \in B\) and \(10 \in B\text{.}\)
