The congruence classes for the relation of congruence modulo 5 on the set of integers are

\begin{align*} \left[ 0 \right] \amp = \left\{ 5n \mid n \in \mathbb{Z} \right\}\\ \left[ 1 \right] \amp = \left\{ 5n +1 \mid n \in \mathbb{Z} \right\}\\ \left[ 2 \right] \amp = \left\{ 5n + 2 \mid n \in \mathbb{Z} \right\}\\ \left[ 3 \right] \amp = \left\{ 5n + 3 \mid n \in \mathbb{Z} \right\}\\ \left[ 4 \right] \amp = \left\{ 5n + 4 \mid n \in \mathbb{Z} \right\} \end{align*}
