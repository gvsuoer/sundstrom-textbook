Let \(A\text{,}\) \(B\text{,}\) and \(C\) be subsets of some universal set \(U\text{.}\) Then all of the following equalities hold.

Properties of the Empty Set and the Universal Set
\begin{align*} A \cap \emptyset \amp = \emptyset \amp A \cap U \amp = A\\ A \cup \emptyset \amp = A \amp A \cup U \amp = U \end{align*}
Idempotent Laws
\begin{align*} A \cap A \amp = A \amp A \cup A \amp = A \end{align*}
Commutative Laws
\begin{align*} A \cap B \amp = B \cap A \amp A \cup B \amp = B \cup A \end{align*}
Associative Laws
\begin{align*} \left( A \cap B \right) \cap C \amp = A \cap \left(B \cap C \right)\\ \left( A \cup B \right) \cup C \amp = A \cup \left(B \cup C \right) \end{align*}
Distributive Laws
\begin{align*} A \cap \left( B \cup C \right) \amp = \left( A \cap B \right) \cup \left( A \cap C \right)\\ A \cup \left( B \cap C \right) \amp = \left( A \cup B \right) \cap \left( A \cup C \right) \end{align*}
