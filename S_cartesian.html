    Section 5.4 Cartesian Products

    Beginning Activity Beginning Activity 1: An Equation with Two Variables
    A4US

    In Section¬†2.3, we introduced the concept of the truth set of an open sentence with one variable. This was defined to be the set of all elements in the universal set that can be substituted for the variable to make the open sentence a true statement.

    In previous mathematics courses, we have also had experience with open sentences with two variables. For example, if we assume that \(x\) and \(y\) represent real numbers, then the equation

    \begin{equation*} 2x + 3y = 12 \end{equation*}

    is an open sentence with two variables. An element of the truth set of this open sentence (also called a solution of the equation) is an ordered pair \(\left( {a,b} \right)\) of real numbers so that when \(a\) is substituted for \(x\) and \(b\) is substituted for \(y\text{,}\) the open sentence becomes a true statement (a true equation in this case). For example, we see that the ordered pair \((6, 0)\) is in the truth set for this open sentence since

    \begin{equation*} 2 \cdot 6 + 3 \cdot 0 = 12 \end{equation*}

    is a true statement. On the other hand, the ordered pair \((4, 1)\) is not in the truth set for this open sentence since

    \begin{equation*} 2 \cdot 4 + 3 \cdot 1 = 12 \end{equation*}

    is a false statement.

    Important Note: The order of the two numbers in the ordered pair is very important. We are using the convention that the first number is to be substituted for \(x\) and the second number is to be substituted for \(y\text{.}\) With this convention, \(\left( 3, 2 \right)\) is a solution of the equation \(2x + 3y = 12\text{,}\) but \(\left( 2, 3 \right)\) is not a solution of this equation.

    1.

    List six different elements of the truth set (often called the solution set) of the open sentence with two variables \(2x + 3y = 12\text{.}\)

    2.

    From previous mathematics courses, we know that the graph of the equation \(2x + 3y = 12\) is a straight line. Sketch the graph of the equation \(2x + 3y = 12\) in the \(xy\)-coordinate plane. What does the graph of the equation \(2x + 3y = 12\) show?

    3.

    Write a description of the solution set \(S\) of the equation \(2x + 3y = 12\) using set builder notation.

    Beginning Activity Beginning Activity 2: The Cartesian Product of Two Sets
    A4US

    In Beginning Activity 1, we worked with ordered pairs without providing a formal definition of an ordered pair. We instead relied on your previous work with ordered pairs, primarily from graphing equations with two variables. Following is a formal definition of an ordered pair.

    Definition.

    Let \(A\) and \(B\) be sets. An ordered pair (with first element from \(A\) and second element from \(B\)) is a single pair of objects, denoted by \((a, b)\) , with \(a \in A\) and \(b \in B\) and an implied order. This means that for two ordered pairs to be equal, they must contain exactly the same objects in the same order. That is, if \(a, c \in A\) and \(b, d \in B\text{,}\) then

    \begin{equation*} \left( {a,b} \right) = \left( {c,d} \right) \text{ if and only if } a = c \text{ and } b = d\text{.} \end{equation*}

    The objects in the ordered pair are called the coordinates of the ordered pair. In the ordered pair \(\left( {a,b} \right)\text{,}\) \(a\) is the first coordinate and \(b\) is the second coordinate.

    We will now introduce a new set operation that gives a way of combining elements from two given sets to form ordered pairs. The basic idea is that we will create a set of ordered pairs.

    Definition.

    If \(A\) and \(B\) are sets, then the Cartesian product, \(A \times B\text{,}\) of \(A\) and \(B\) is the set of all ordered pairs \(\left( {x,y} \right)\) where \(x \in A\) and \(y \in B\text{.}\) We use the notation \(A \times B\) for the Cartesian product of \(A\) and \(B\text{,}\) and using set builder notation, we can write \(A \times B = \left\{ { {\left( {x,y} \right)} \mid x \in A\text{ and } y \in B} \right\}\text{.}\) We frequently read \(A \times B\) as ‚Äú\(A\) cross \(B\text{.}\)‚Äù In the case where the two sets are the same, we will write \(A^2\) for \(A \times A\text{.}\) That is,

    \begin{equation*} A^2 = A \times A = \left\{ {\left( {a,b} \right) \mid a \in A\text{ and } b \in A} \right\}\text{.} \end{equation*}

    Let \(A = \left\{ {1,2,3} \right\}\) and \(B = \left\{ {a,b} \right\}\text{.}\)

    1.

    Is the ordered pair \(\left( {3,a} \right)\) in the Cartesian product \(A \times B\text{?}\) Explain.

    2.

    Is the ordered pair \(\left( {3,a} \right)\) in the Cartesian product \(A \times A\text{?}\) Explain.

    3.

    Is the ordered pair \(\left( {3,1} \right)\) in the Cartesian product \(A \times A\text{?}\) Explain.

    4.

    Subsection Cartesian Products

    When working with Cartesian products, it is important to remember that the Cartesian product of two sets is itself a set. As a set, it consists of a collection of elements. In this case, the elements of a Cartesian product are ordered pairs. We should think of an ordered pair as a single object that consists of two other objects in a specified order. For example,

    • If \(a \ne 1\text{,}\) then the ordered pair \(\left( {1,a} \right)\) is not equal to the ordered pair \(\left( {a,1} \right)\text{.}\) That is, \(\left( {1,a} \right) \ne \left( {a,1} \right)\text{.}\)

    • If \(A = \left\{ {1,2,3} \right\}\) and \(B = \left\{ {a,b} \right\}\text{,}\) then the ordered pair \(\left( {3,a} \right)\) is an element of the set \(A \times B\text{.}\) That is, \(\left( {3,a} \right) \in A \times B\text{.}\)

    • If \(A = \left\{ {1,2,3} \right\}\) and \(B = \left\{ {a,b} \right\}\text{,}\) then the ordered pair \(\left( {5,a} \right)\) is not an element of the set \(A \times B\) since \(5 \notin A\text{.}\) That is, \(\left( {5,a} \right) \notin A \times B\text{.}\)

    In Section¬†5.3, we studied certain properties of set union, set intersection, and set complements, which we called the algebra of sets. We will now begin something similar for Cartesian products. We begin by examining some specific examples in Progress Check¬†5.30 and a little later in Progress Check¬†5.32.

    Subsection The Cartesian Plane

    In Beginning Activity 1, we sketched the graph of the equation \(2x + 3y = 12\) in the \(xy\)-plane. This \(xy\)-plane, with which you are familiar, is a representation of the set \(\mathbb{R} \times \mathbb{R}\) or \(\mathbb{R}^2\text{.}\) This plane is called the Cartesian plane.

    The basic idea is that each ordered pair of real numbers corresponds to a point in the plane, and each point in the plane corresponds to an ordered pair of real numbers. This geometric representation of \(\mathbb{R}^2\) is an extension of the geometric representation of \(\mathbb{R}\) as a straight line whose points correspond to real numbers.

    Since the Cartesian product \(\mathbb{R}^2\) corresponds to the Cartesian plane, the Cartesian product of two subsets of \(\mathbb{R}\) corresponds to a subset of the Cartesian plane. For example, if \(A\) is the interval \(\left[ {1,3} \right]\text{,}\) and \(B\) is the interval \(\left[ {2,5} \right]\text{,}\) then

    \begin{equation*} A \times B = \left\{ {\left( {x, y} \right) \in \mathbb{R}^2 \mid 1 \leq x \leq 3\text{ and } 2 \leq y \leq 5} \right\}\!\text{.} \end{equation*}

    A graph of the set \(A \times B\) can then be drawn in the Cartesian plane as shown in Figure¬†5.31.

    This illustrates that the graph of a Cartesian product of two intervals of finite length in \(\mathbb{R}\) corresponds to the interior of a rectangle and possibly some or all of its boundary. The solid line for the boundary in Figure¬†5.31 indicates that the boundary is included. In this case, the Cartesian product contained all of the boundary of the rectangle. When the graph does not contain a portion of the boundary, we usually draw that portion of the boundary with a dotted line.

    Note: A Caution about Notation. The standard notation for an open interval in \(\mathbb{R}\) is the same as the notation for an ordered pair, which is an element of \mbox{\(\mathbb{R} \times \mathbb{R}\)} . We need to use the context in which the notation is used to determine which interpretation is intended. For example,

    • If we write \(\left( {\sqrt 2 ,7} \right) \in \mathbb{R} \times \mathbb{R}\text{,}\) then we are using \(\left( {\sqrt 2 ,7} \right)\) to represent an ordered pair of real numbers.

    • If we write \(\left( {1,2} \right) \times \left\{ 4 \right\}\text{,}\) then we are interpreting \(\left( {1,2} \right)\) as an open interval. We could write

      \begin{equation*} \left( {1,2} \right) \times \left\{ 4 \right\} = \left\{ {\left. {\left( {x,4} \right)\,} \right|1 \lt x \lt 2} \right\}\!\text{.} \end{equation*}

    One purpose of the work in Progress Check¬†5.30 and Progress Check¬†5.32 was to indicate the plausibility of many of the results contained in the next theorem.

    Theorem 5.33.

    Let \(A\text{,}\) \(B\text{,}\) and \(C\) be sets. Then

    1. \(\displaystyle A \times \left( {B \cap C} \right) = \left( {A \times B} \right) \cap \left( {A \times C} \right)\)

    2. \(\displaystyle A \times \left( {B \cup C} \right) = \left( {A \times B} \right) \cup \left( {A \times C} \right)\)

    3. \(\displaystyle \left( {A \cap B} \right) \times C = \left( {A \times C} \right) \cap \left( {B \times C} \right)\)

    4. \(\displaystyle \left( {A \cup B} \right) \times C = \left( {A \times C} \right) \cup \left( {B \times C} \right)\)

    5. \(\displaystyle A \times \left( {B - C} \right) = \left( {A \times B} \right) - \left( {A \times C} \right)\)

    6. \(\displaystyle \left( {A - B} \right) \times C = \left( {A \times C} \right) - \left( {B \times C} \right)\)

    7. If \(T \subseteq A\text{,}\) then \(T \times B \subseteq A \times B\text{.}\)

    8. If \(Y \subseteq B\text{,}\) then \(A \times Y \subseteq A \times B\text{.}\)

    We will not prove all these results; rather, we will prove Item¬†2 of Theorem¬†5.33 and leave some of the rest to the exercises. In constructing these proofs, we need to keep in mind that Cartesian products are sets, and so we follow many of the same principles to prove set relationships that were introduced in Section¬†5.2 and Section¬†5.3.

    The other thing to remember is that the elements of a Cartesian product are ordered pairs. So when we start a proof of a result such as Item¬†2 of Theorem¬†5.33, the primary goal is to prove that the two sets are equal. We will do this by proving that each one is a subset of the other one. So if we want to prove that \(A \times \left( {B \cup C} \right) \subseteq \left( {A \times B} \right) \cup \left( {A \times C} \right)\text{,}\) we can start by choosing an arbitrary element of \(A \times \left( {B \cup C} \right)\text{.}\) The goal is then to show that this element must be in \(\left( {A \times B} \right) \cup \left( {A \times C} \right)\text{.}\) When we start by choosing an arbitrary element of \(A \times \left( {B \cup C} \right)\text{,}\) we could give that element a name. For example, we could start by letting

    \begin{equation} u \text{ be an element of } A \times \left( {B \cup C} \right)\!\text{.}\tag{47} \end{equation}

    We can then use the definition of ‚Äúordered pair‚Äù to conclude that

    \begin{equation} \text{ there exists } x \in A \text{ and there exists } y \in B \cup C \text{ such that } u = \left( {x,y} \right)\text{.}\tag{48} \end{equation}

    In order to prove that \(A \times \left( {B \cup C} \right) \subseteq \left( {A \times B} \right) \cup \left( {A \times C} \right)\text{,}\) we must now show that the ordered pair \(u\) from (1) is in \(\left( {A \times B} \right) \cup \left( {A \times C} \right)\text{.}\) In order to do this, we can use the definition of set union and prove that

    \begin{equation} u \in (A \times B) \text{ or } u \in (A \times C)\text{.}\tag{49} \end{equation}

    Since \(u = (x, y)\text{,}\) we can prove (49) by proving that

    \begin{equation} \left( x \in A \text{ and } y \in B \right) \text{ or } \left( x \in A \text{ and } y \in C \right)\text{.}\tag{50} \end{equation}

    If we look at the sentences in (48) and (50), it would seem that we are very close to proving that \(A \times \left( {B \cup C} \right) \subseteq \left( {A \times B} \right) \cup \left( {A \times C} \right)\text{.}\) Following is a proof of Item¬†2 of Theorem¬†5.33.

    Theorem 5.34. Item¬†2 of Theorem¬†5.33.

    Let \(A\text{,}\) \(B\text{,}\) and \(C\) be sets. Then

    \begin{equation*} A \times \left( {B \cup C} \right) = \left( {A \times B} \right) \cup \left( {A \times C} \right)\text{.} \end{equation*}

    Proof.

    Let \(A\text{,}\) \(B\text{,}\) and \(C\) be sets. We will prove that \(A \times \left( {B \cup C} \right)\) is equal to \(\left( {A \times B} \right) \cup \left( {A \times C} \right)\) by proving that each set is a subset of the other set.

    To prove that \(A \times \left( {B \cup C} \right) \subseteq \left( {A \times B} \right) \cup \left( {A \times C} \right)\text{,}\) we let \(u \in A \times \left( {B \cup C} \right)\text{.}\) Then there exists \(x \in A\) and there exists \(y \in B \cup C\) such that \(u = \left( {x,y} \right)\text{.}\) Since \(y \in B \cup C\text{,}\) we know that \(y \in B\) or \(y \in C\text{.}\)

    In the case where \(y \in B\text{,}\) we have \(u = \left( {x,y} \right)\text{,}\) where \(x \in A\) and \(y \in B\text{.}\) So in this case, \(u \in A \times B\text{,}\) and hence \(u \in \left( {A \times B} \right) \cup \left( {A \times C} \right)\text{.}\) Similarly, in the case where \(y \in C\text{,}\) we have \(u = \left( {x,y} \right)\text{,}\) where \(x \in A\) and \(y \in C\text{.}\) So in this case, \(u \in A \times C\) and, hence, \(u \in \left( {A \times B} \right) \cup \left( {A \times C} \right)\text{.}\)

    In both cases, \(u \in \left( {A \times B} \right) \cup \left( {A \times C} \right)\text{.}\) Hence, we may conclude that if \(u\) is an element of \(A \times \left( {B \cup C} \right)\text{,}\) then \(u \in \left( {A \times B} \right) \cup \left( {A \times C} \right)\text{,}\) and this proves that

    \begin{equation} A \times \left( {B \cup C} \right) \subseteq \left( {A \times B} \right) \cup \left( {A \times C} \right)\!\text{.}\tag{51} \end{equation}

    We must now prove that \(\left( {A \times B} \right) \cup \left( {A \times C} \right) \subseteq A \times \left( {B \cup C} \right)\text{.}\) So we let \(v \in \left( {A \times B} \right) \cup \left( {A \times C} \right)\text{.}\) Then \(v \in \left( {A \times B} \right)\) or \(v \in \left( {A \times C} \right)\text{.}\)

    In the case where \(v \in \left( {A \times B} \right)\text{,}\) we know that there exists \(s \in A\) and there exists \(t \in B\) such that \(v = \left( {s,\;t} \right)\text{.}\) But since \(t \in B\text{,}\) we know that \(t \in B \cup C\text{,}\) and hence \(v \in A \times \left( {B \cup C} \right)\text{.}\) Similarly, in the case where \(v \in \left( {A \times C} \right)\text{,}\) we know that there exists \(s \in A\) and there exists \(t \in C\) such that \(v = \left( {s,t} \right)\text{.}\) But because \(t \in C\text{,}\) we can conclude that \(t \in B \cup C\) and, hence, \(v \in A \times \left( {B \cup C} \right)\text{.}\)

    In both cases, \(v \in A \times \left( {B \cup C} \right)\text{.}\) Hence, we may conclude that if

    \(v \in \left( {A \times B} \right) \cup \left( {A \times C} \right)\text{,}\) then \(v \in A \times \left( {B \cup C} \right)\text{,}\) and this proves that

    \begin{equation} \left( {A \times B} \right) \cup \left( {A \times C} \right) \subseteq A \times \left( {B \cup C} \right)\!\text{.}\tag{52} \end{equation}

    The relationships in (51) and (52) prove that \(A \times \left( {B \cup C} \right) = \left( {A \times B} \right) \cup \left( {A \times C} \right)\text{.}\)

    Activity 32. A Set Theoretic Definition of an Ordered Pair.

    In elementary mathematics, the notion of an ordered pair introduced at the beginning of this section will suffice. However, if we are interested in a formal development of the Cartesian product of two sets, we need a more precise definition of ordered pair. Following is one way to do this in terms of sets. This definition is credited to Kazimierz Kuratowski (1896 ‚Äî 1980). Kuratowski was a famous Polish mathematician whose main work was in the areas of topology and set theory. He was appointed the Director of the Polish Academy of Sciences and served in that position for 19 years. Let \(x\) be an element of the set \(A\text{,}\) and let \(y\) be an element of the set \(B\text{.}\) The ordered pair \(\left( {x,y} \right)\) is defined to be the set \(\left\{ {\left\{ x \right\},\left\{ {x,y} \right\}} \right\}\text{.}\) That is,

    \begin{equation*} \left( {x,y} \right) = \left\{ {\left\{ x \right\},\left\{ {x,y} \right\}} \right\}\!\text{.} \end{equation*}
