Definition.

Let \(f\x S \to T\text{.}\) If \(C \subseteq T\text{,}\) then the preimage of \(\boldsymbol{C}\) under \(\boldsymbol{f}\) is the set \(f^{-1} ( C )\text{,}\) where

\begin{equation*} f^{-1} ( C ) = \left\{x \in S \mid f ( x ) \in C \right\}\text{.} \end{equation*}

If there is no confusion as to which function is being used, we call \(f^{-1} ( C )\) the preimage of \(\boldsymbol{C}\). The preimage of the set \(C\) under \(f\) is also called the inverse image of \(\boldsymbol{C}\) under \(\boldsymbol{f}\).
