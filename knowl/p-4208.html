Let \(A = \left\{ {1, 2, 3} \right\}\) and let \(C = \left\{ {a, b, c, d} \right\}\text{.}\) Define the function \(g\x A \to C\) by \(g( 1 ) = a\text{,}\) \(g( 2 ) = b\text{,}\) and \(g( 3 ) = d\text{.}\) Write the function \(g\) as a set of ordered pairs in \(A \times C\text{.}\)
