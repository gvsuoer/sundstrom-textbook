In each of the indexed families of sets that we seen so far, if the indices were different, then the sets were different. That is, if \(\Lambda\) is an indexing set for the family of sets \(\mathscr{A} = \left\{ A_\alpha \mid \alpha \in \Lambda \right\}\text{,}\) then if \(\alpha, \beta \in \Lambda\) and \(\alpha \ne \beta\text{,}\) then \(A_\alpha \ne A_\beta\text{.}\) (Note: The letter \(\beta\) is the Greek lowercase beta.)
