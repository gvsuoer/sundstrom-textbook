Some of the properties of set operations are closely related to some of the logical operators we studied in SectionÂ 2.1. This is due to the fact that set intersection is defined using a conjunction (and), and set union is defined using a disjunction (or). For example, if \(A\) and \(B\) are subsets of some universal set \(U\text{,}\) then an element \(x\) is in \(A \cup B\) if and only if \(x \in A\) or \(x \in B\text{.}\)
