Some mathematical results are stated in the form “\(P\) if and only if \(Q\)” or “\(P\) is necessary and sufficient for \(Q\text{.}\)” An example would be, “A triangle is equilateral if and only if its three interior angles are congruent.” The symbolic form for the biconditional statement “\(P\) if and only if \(Q\)” is \(P \leftrightarrow Q\text{.}\) In order to determine a truth table for a biconditional statement, it is instructive to look carefully at the form of the phrase “\(P\) if and only if \(Q\text{.}\)” The word “and” suggests that this statement is a conjunction. Actually it is a conjunction of the statements “\(P\) if \(Q\)” and “\(P\) only if \(Q\text{.}\)” The symbolic form of this conjunction is \(\left[ {\left( {Q \to P} \right) \wedge \left( {P \to Q} \right)} \right]\text{.}\)
