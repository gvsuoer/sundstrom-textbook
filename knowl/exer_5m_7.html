Exercise 4.

Construct a know-show table and write a complete proof for each of the following statements:

(a)

If \(m\) is an even integer, then \(5m + 7\) is an odd integer.

Answer.
(b)

If \(m\) is an odd integer, then \(5m + 7\) is an even integer.

(c)

If \(m\) and \(n\) are odd integers, then \(mn + 7\) is an even integer.
