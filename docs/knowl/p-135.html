Consider the following sentence:

If \(x\) is a positive real number, then \(x^2 + 8x\) is a positive real number.
Although the hypothesis and conclusion of this conditional sentence are not statements, the conditional sentence itself can be considered to be a statement as long as we know what possible numbers may be used for the variable \(x\text{.}\) From the context of this sentence, it seems that we can substitute any positive real number for \(x\text{.}\) We can also substitute 0 for \(x\) or a negative real number for \(x\) provided that we are willing to work with a false hypothesis in the conditional statement.
