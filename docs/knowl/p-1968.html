Suppose we want to determine the possible values for \(a^2\) modulo 6 for odd integers that are not multiples of 3. Before beginning to use congruence arithmetic (as in the proof of PropositionÂ 3.39) in each of the possible six cases, we can show that some of the cases are not possible under these assumptions. (In some sense, we use a short proof by contradiction for these cases.) So assume that \(a\) is an odd integer. Then:

  • If \(a \equiv 0 \pmod 6\text{,}\) then there exists an integer \(k\) such that \(a = 6k\text{.}\) But then \(a = 2(3k)\) and hence, \(a\) is even. Since we assumed that \(a\) is odd, this case is not possible.

  • If \(a \equiv 2 \pmod 6\text{,}\) then there exists an integer \(k\) such that \(a = 6k + 2\text{.}\) But then \(a = 2(3k + 1)\) and hence, \(a\) is even. Since we assumed that \(a\) is odd, this case is not possible.
