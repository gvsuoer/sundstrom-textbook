One reason for the definition of proper subset is that each set is a subset of itself. That is, If \(A\) is a set, then \(A \subseteq A\text{.}\) However, sometimes we need to indicate that a set \(X\) is a subset of \(Y\) but \(X \ne Y\text{.}\) For example, if

\begin{equation*} X = \left\{ {1, 2} \right\}\text{ and } Y = \left\{ {0, 1, 2, 3} \right\}\!\text{,} \end{equation*}

then \(X \subset Y\text{.}\) We know that \(X \subseteq Y\) since each element of \(X\) is an element of \(Y\text{,}\) but \(X \ne Y\) since \(0 \in Y\) and \(0 \notin X\text{.}\) (Also, \(3 \in Y\) and \(3 \notin X\text{.}\)) Notice that the notations \(A \subset B\) and \(A \subseteq B\) are used in a manner similar to inequality notation for numbers (\(a \lt b\) and \(a \leq b\)).
