Theorem 3.8.

For each integer \(n\text{,}\) if \(n^2\) is an even integer, then \(n\) is an even integer.

Proof.

We will prove this result by proving the contrapositive of the statement, which is

For each integer \(n\text{,}\) if \(n\) is an odd integer, then \(n^2\) is an odd integer.

However, in TheoremÂ 1.10, we have already proven that if \(x\) and \(y\) are odd integers, then \(x \cdot y\) is an odd integer. So using \(x = y = n\text{,}\) we can conclude that if \(n\) is an odd integer, then \(n \cdot n\text{,}\) or \(n^2\text{,}\) is an odd integer. We have thus proved the contrapositive of the theorem, and consequently, we have proved that if \(n^2\) is an even integer, then \(n\) is an even integer.
