Item 1.

\(\left( {\forall x \in \mathbb{Z}} \right)\left( {\forall y \in \mathbb{Z}} \right) \left( {x + y = 0} \right)\)

We could read this as, “For all integers \(x\) and \(y\text{,}\) \(x + y = 0\text{.}\)” This is a false statement since it is possible to find two integers whose sum is not zero \(\left( {2 + 3 \ne 0} \right)\text{.}\)
