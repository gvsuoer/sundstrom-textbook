In our discussion of the power set, we were concerned with the number of elements in a set. In fact, the number of elements in a finite set is a distinguishing characteristic of the set, so we give it the following name.

Definition.

The number of elements in a finite set \(A\) is called the cardinality of \(A\) and is denoted by \(\card(A)\text{.}\)

For example, \(\card (\emptyset) = 0\text{;}\) \(\card (\left\{ {a,b} \right\}) = 2\text{;}\) \(\card \left( \mathcal{P}( \left\{ {a,b} \right\}) \right) = 4\text{.}\)
