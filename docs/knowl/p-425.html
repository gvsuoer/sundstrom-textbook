An integer \(a\) is said to be a type 0 integer if there exists an integer \(n\) such that \(a=3n\text{.}\) An integer \(a\) is said to be a type 1 integer if there exists an integer \(n\) such that \(a=3n+1\text{.}\) An integer \(a\) is said to be a type 2 integer if there exists an integer \(m\) such that \(a=3m+2\text{.}\)
