Section 2.3 Open Sentences and Sets

Beginning Activity Beginning Activity 1: Sets and Set Notation
A4US

The theory of sets is fundamental to mathematics in the sense that many areas of mathematics use set theory and its language and notation. This language and notation must be understood if we are to communicate effectively in mathematics. At this point, we will give a very brief introduction to some of the terminology used in set theory.

A set is a well-defined collection of objects that can be thought of as a single entity itself. For example, we can think of the set of integers that are greater than 4. Even though we cannot write down all the integers that are in this set, it is still a perfectly well-defined set. This means that if we are given a specific integer, we can tell whether or not it is in the set of integers greater than 4.

The most basic way of specifying the elements of a set is to list the elements of that set. This works well when the set contains only a small number of objects. The usual practice is to list these elements between braces. For example, if the set \(C\) consists of the integer solutions of the equation \(x^2 = 9\text{,}\) we would write

\begin{equation*} C = \left\{ { - 3,3} \right\}\text{.} \end{equation*}

For larger sets, it is sometimes inconvenient to list all of the elements of the set. In this case, we often list several of them and then write a series of three dots  (\(\ldots\)) to indicate that the pattern continues. For example,

\begin{equation*} D = \left\{ 1, 3, 5, 7, \ldots, 49 \right\} \end{equation*}

is the set of all odd natural numbers from 1 to 49, inclusive.

For some sets, it is not possible to list all of the elements of a set; we then list several of the elements in the set and again use a series of three dots  (\(\ldots\)) to indicate that the pattern continues. For example, if \(F\) is the set of all even natural numbers, we could write

\begin{equation*} F = \left\{ {2,4,6, \ldots } \right\}\text{.} \end{equation*}

We can also use the three dots before listing specific elements to indicate the pattern prior to those elements. For example, if \(E\) is the set of all even integers, we could write

\begin{equation*} E = \left\{ {\ldots -6, -4, -2, 0, 2,4,6, \ldots } \right\}\text{.} \end{equation*}

Listing the elements of a set inside braces is called the roster method of specifying the elements of the set. We will learn other ways of specifying the elements of a set later in this section.

Not all mathematical sentences are statements. For example, an equation such as

\begin{equation*} x^2 - 5 = 0 \end{equation*}

is not a statement. In this sentence, the symbol \(x\) is a variable. It represents a number that may be chosen from some specified set of numbers. The sentence (equation) becomes true or false when a specific number is substituted for \(x\text{.}\)

1.

Does the equation \(x^2 - 25 = 0\) become a true statement...

(a)

if \(- 5\) is substituted for \(x\text{?}\)

(b)

if \(\sqrt{5}\) is substituted for \(x\text{?}\)

Definition 2.16.

A variable is a symbol representing an unspecified object that can be chosen from a given set \(U\text{.}\) The set \(U\) is called the universal set for the variable. It is the set of specified objects from which objects may be chosen to substitute for the variable. A constant is a specific member of the universal set.

Some sets that we will use frequently are the usual number systems. Recall that we use the symbol \(\mathbb{R}\) to stand for the set of all real numbers, the symbol \(\mathbb{Q}\) to stand for the set of all rational numbers, the symbol \(\mathbb{Z}\) to stand for the set of all integers, and the symbol \(\mathbb{N}\) to stand for the set of all natural numbers.

Subsection Some Set Notation

In Beginning Activity 1, we indicated that a set is a well-defined collection of objects that can be thought of as an entity itself.

When working with a mathematical object, such as set, we need to define when two of these objects are equal. We are also often interested in whether or not one set is contained in another set.

Definition 2.18.

Two sets, \(A\) and \(B\text{,}\) are equal when they have precisely the same elements. In this case, we write \(A = B\) . When the sets \(A\) and \(B\) are not equal, we write \(A \ne B\text{.}\)

The set \(A\) is a subset of a set \(B\) provided that each element of \(A\) is an element of \(B\text{.}\) In this case, we write \(A \subseteq B\) and also say that \(A\) is contained in \(B\text{.}\) When \(A\) is not a subset of \(B\text{,}\) we write \(A \not \subseteq B\text{.}\)

Using these definitions, we see that for any set \(A\text{,}\) \(A = A\) and since it is true that for each \(x \in U\text{,}\) if \(x \in A\text{,}\) then \(x \in A\text{,}\) we also see that \(A \subseteq A\text{.}\) That is, any set is equal to itself and any set is a subset of itself. For some specific examples, we see that:

  • \(\displaystyle \left\{ {1, 3, 5} \right\} = \left\{ {3, 5, 1} \right\}\)

  • \(\displaystyle \left\{ {4, 8, 12} \right\} = \left\{ {4, 4, 8, 12, 12} \right\}\)

  • \(\displaystyle \left\{ {5, 10} \right\} = \left\{ {5, 10, 5} \right\}\)

  • \(\displaystyle \left\{ {5, 10} \right\} \ne \left\{ {5, 10, 15} \right\} \text{ but } \left\{ {5, 10} \right\} \subseteq \left\{ {5, 10, 15} \right\} \text{ and } \left\{ {5, 10, 15} \right\} \not \subseteq \left\{ {5, 10} \right\}\)

In each of the first three examples, the two sets have exactly the same elements even though the elements may be repeated or written in a different order.

Subsection Variables and Open Sentences

As we have seen in the beginning activities, not all mathematical sentences are statements. This is often true if the sentence contains a variable. The following terminology is useful in working with sentences and statements.

Definition 2.20.

An open sentence is a sentence \(P( {x_1 ,x_2 , \ldots ,x_n })\) involving variables \(x_1 ,x_2 , \ldots ,x_n\) with the property that when specific values from the universal set are assigned to \(x_1 ,x_2 , \ldots ,x_n\text{,}\) then the resulting sentence is either true or false. That is, the resulting sentence is a statement. An open sentence is also called a predicate or a propositional function.

Notation.

One reason an open sentence is sometimes called a propositional function is the fact that we use function notation \(P( {x_1 ,x_2 , \ldots ,x_n })\) for an open sentence in \(n\) variables. When there is only one variable, such as \(x\text{,}\) we write \(P( x )\text{,}\) which is read “\(P\) of \(x\text{.}\)” In this notation, \(x\) represents an arbitrary element of the universal set, and \(P(x)\) represents a sentence. When we substitute a specific element of the universal set for \(x\text{,}\) the resulting sentence becomes a statement. This is illustrated in the next example.

Example 2.21. Open Sentences.

If the universal set is \(\mathbb{R}\text{,}\) then the sentence “\(x^2 - 3x - 10 = 0\)” is an open sentence involving the one variable \(x\text{.}\)

If we substitute \(x = 2\text{,}\) we obtain the false statement “\(2^2 - 3 \cdot 2 - 10 = 0\text{.}\)”
If we substitute \(x = 5\text{,}\) we obtain the true statement “\(5^2 - 3 \cdot 5 - 10 = 0\text{.}\)”
In this example, we can let \(P( x )\) be the predicate “\(x^2 - 3x - 10 = 0\)” and then say that \(P( 2 )\) is false and \(P( 5 )\) is true.

Using similar notation, we can let \(Q( {x, y} )\) be the predicate “\(x + 2y = 7\text{.}\)” This predicate involves two variables. Then,

\(Q( {1, 1})\) is false since “\(1 + 2 \cdot 1 = 7\)” is false; and
\(Q( {3, 2} )\) is true since “\(3 + 2 \cdot 2 = 7\)” is true.

Without using the term, Example 2.21 and Progress Check 2.22 (and Beginning Activity 2) dealt with a concept called the truth set of a predicate.

Definition 2.23.

The truth set of an open sentence with one variable is the collection of objects in the universal set that can be substituted for the variable to make the predicate a true statement.

One part of elementary mathematics consists of learning how to solve equations. In more formal terms, the process of solving an equation is a way to determine the truth set for the equation, which is an open sentence. In this case, we often call the truth set the solution set. Following are three examples of truth sets.

  • If the universal set is \(\R\text{,}\) then the truth set of the equation \(3x - 8 = 10\) is the set \(\{ 6 \}\text{.}\)

  • If the universal set is \(\mathbb{R}\text{,}\) then the truth set of the equation “\(x^2 - 3x - 10 = 0\)” is \(\left\{ { - 2, 5} \right\}\text{.}\)

  • If the universal set is \(\mathbb{N}\text{,}\) then the truth set of the open sentence “\(\sqrt {n} \in \mathbb{N}\)” is \(\left\{ {1, 4, 9, 16, \ldots } \right\}\text{.}\)

Subsection Set Builder Notation

Sometimes it is not possible to list all the elements of a set. For example, if the universal set is \(\mathbb{R}\text{,}\) we cannot list all the elements of the truth set of “\(x^2 \lt 4\text{.}\)” In this case, it is sometimes convenient to use the so-called set builder notation in which the set is defined by stating a rule that all elements of the set must satisfy. If \(P(x)\) is a predicate in the variable \(x\text{,}\) then the notation

\begin{equation*} \left\{ x \in U \mid P(x) \right\} \end{equation*}

stands for the set of all elements \(x\) in the universal set \(U\) for which \(P(x)\) is true. If it is clear what set is being used for the universal set, this notation is sometimes shortened to \(\left\{ x \mid {P(x)} \right\}\text{.}\) This is usually read as “the set of all \(x\) such that \(P(x)\text{.}\)” The vertical bar stands for the phrase “such that.” Some writers will use a colon (:) instead of the vertical bar.

For a non-mathematical example, \(P\) could be the property that a college student is a mathematics major. Then \(\left\{ x \mid {P(x)} \right\}\) denotes the set of all college students who are mathematics majors. This could be written as

\begin{equation*} \left\{ x \mid x\text{ is a college student who is a mathematics major } \right\}\text{.} \end{equation*}
Example 2.24. Truth Sets.

Assume the universal set is \(\R\) and \(P( x )\) is “\(x^2 \lt 4\text{.}\)” We can describe the truth set of \(P(x)\) as the set of all real numbers whose square is less than 4. We can also use set builder notation to write the truth set of \(P( x )\) as

\begin{equation*} \left\{ {x \in \R} \mid x^2 \lt 4 \right\}\text{.} \end{equation*}

However, if we solve the inequality \(x^2 \lt 4\text{,}\) we obtain \(- 2 \lt x \lt 2\text{.}\) So we could also write the truth set as

\begin{equation*} \left\{ {x \in \R} \mid - 2 \lt x \lt 2 \right\}\text{.} \end{equation*}

We could read this as the set of all real numbers that are greater than \(-2\) and less than 2. We can also write

\begin{equation*} \left\{ {x \in \R} \mid x^2 \lt 4 \right\} = \left\{ {x \in \mathbb{R}} \mid - 2 \lt x \lt 2 \right\}\text{.} \end{equation*}
Progress Check 2.25. Working with Truth Sets.

Let \(P( x )\) be the predicate “\(x^2 \leq 9\text{.}\)”

(a)

If the universal set is \(\mathbb{R}\text{,}\) describe the truth set of \(P(x)\) using English and write the truth set of \(P( x )\) using set builder notation.

Solution.

The truth set is the set of all real numbers whose square is less than or equal to 9. The truth set is \(\left\{ {\left. {x \in \mathbb{R} } \right| x^2 \leq 9} \right\} = \left\{ \left. x \in \R \right| -3 \leq x \leq 3 \right\}\text{.}\)

(b)

If the universal set is \(\mathbb{Z}\text{,}\) then what is the truth set of \(P( x )\text{?}\) Describe this set using English and then use the roster method to specify all the elements of this truth set.

Solution.

The truth set is the set of all integers whose square is less than or equal to 9. The truth set is \(\left\{ { - 3, - 2, - 1, 0, 1, 2, 3} \right\}\text{.}\)

(c)

Are the truth sets in Task 2.25.a and Task 2.25.b equal? Explain.

Solution.

The truth sets in Parts (1) and (2) equal are not equal. One purpose of this progress check is to show that the truth set of a predicate depends on the predicate and on the universal set.

So far, our standard form for set builder notation has been \(\left\{ x \in U \mid P( x ) \right\}\text{.}\) It is sometimes possible to modify this form and put the predicate first. For example, the set

\begin{equation*} A = \left\{ 3n + 1 \mid n \in \N \right\} \end{equation*}

describes the set of all natural numbers of the form \(3n + 1\) for some natural number. By substituting 1, 2, 3, 4, and so on, for \(n\text{,}\) we can use the roster method to write

\begin{equation*} A = \left\{ 3n + 1 \mid n \in \N \right\} = \left\{4, 7, 10, 13, \ldots \right\}\text{.} \end{equation*}

We can sometimes “reverse this process” by starting with a set specified by the roster method and then writing the same set using set builder notation.

Example 2.26. Set Builder Notation.

Let \(B = \left\{ \ldots, -11, -7, -3, 1, 5, 9, 13, \ldots \right\}\text{.}\) The key to writing this set using set builder notation is to recognize the pattern involved. We see that once we have an integer in \(B\text{,}\) we can obtain another integer in \(B\) by adding 4. This suggests that the predicate we will use will involve multiplying by 4.

Since it is usually easier to work with positive numbers, we notice that \(1 \in B\) and \(5 \in B\text{.}\) Notice that

\begin{equation*} 1 = 4 \cdot 0 + \text{ and } 5 = 4 \cdot 1 + 1\text{.} \end{equation*}

This suggests that we might try \(\left\{ 4n + 1 \mid n \in \Z \right\}\text{.}\) In fact, by trying other integers for \(n\text{,}\) we can see that

\begin{equation*} B = \left\{ \ldots, -11, -7, -3, 1, 5, 9, 13, \ldots \right\} = \left\{ 4n + 1 \mid n \in \Z \right\}\text{.} \end{equation*}

Determine four elements of each set other than the ones listed using the roster method.

(b)

Use set builder notation to describe each set.

\begin{align*} A \amp = \left\{ 4n - 3 \mid n \in \N \right\} = \left\{ x \in \N \mid x = 4n - 3 \text{ for some natural number } n \right\}.\\ B \amp = \left\{ -2n \mid n \text{ is a nonnegative integer } \right\}.\\ C \amp = \left\{ \left( \sqrt{2} \right)^{2m - 1} \mid m \in \N \right\} = \left\{ \left( \sqrt{2} \right)^{n} \mid n \text{ is an odd natural number } \right\}.\\ D \amp = \left\{ 3^n \mid n \text{ is a nonnegative integer } \right\}\text{.} \end{align*}

Subsection The Empty Set

When a set contains no elements, we say that the set is the empty set. For example, the set of all rational numbers that are solutions of the equation \(x^2 = -2\) is the empty set since this equation has no solutions that are rational numbers.

In mathematics, the empty set is usually designated by the symbol \(\emptyset\text{.}\) We usually read the symbol \(\emptyset\) as “the empty set” or “the null set.” (The symbol \(\emptyset\) is actually the next to last letter in the Danish-Norwegian alphabet.)

Subsection When the Truth Set Is the Universal Set

The truth set of a predicate can be the universal set. For example, if the universal set is the set of real numbers \(\mathbb{R}\text{,}\) then the truth set of the predicate  “\(x + 0 = x\)” is \(\mathbb{R}\text{.}\)

Notice that the sentence “\(x + 0 = x\)” has not been quantified and a particular element of the universal set has not been substituted for the variable \(x\text{.}\) Even though the truth set for this sentence is the universal set, we will adopt the convention that unless the quantifier is stated explicitly, we will consider the sentence to be a predicate or open sentence. So, with this convention, if the universal set is \(\mathbb{R}\text{,}\) then

  • \(x + 0 = x\) is a predicate;

  • For each real number \(x\text{,}\) \(\left( x + 0 = x \right)\) is a statement.

Activity 7. Closure Explorations.

In Section 1.1, we studied some of the closure properties of the standard number systems. (See Closure Properties of Number Systems.) We can extend this idea to other sets of numbers. So we say that:

  • A set \(A\) of numbers is closed under addition provided that whenever \(x\) and \(y\) are in the set \(A\text{,}\) \(x + y\) is in the set \(A\text{.}\)

  • A set \(A\) of numbers is closed under multiplication provided that whenever \(x\) and \(y\) are are in the set \(A\text{,}\) \(x \cdot y\) is in the set \(A\text{.}\)

  • A set \(A\) of numbers is closed under subtraction provided that whenever \(x\) and \(y\) are are in the set \(A\text{,}\) \(x - y\) is in the set \(A\text{.}\)
