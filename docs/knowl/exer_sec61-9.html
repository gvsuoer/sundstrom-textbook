Recall that a real function is a function whose domain and codomain are subsets of the real numbers \(\R\text{.}\) (See The Graph of a Real Function.) Most of the functions used in calculus are real functions. Quite often, a real function is given by a formula or a graph with no specific reference to the domain or the codomain. In these cases, the usual convention is to assume that the domain of the real function \(f\) is the set of all real numbers \(x\) for which \(f( x )\) is a real number, and that the codomain is \(\mathbb{R}\text{.}\) For example, if we define the (real) function \(f\) by

\begin{equation*} f( x ) = \frac{x}{{x - 2}}\text{,} \end{equation*}

we would be assuming that the domain is the set of all real numbers that are not equal to 2 and that the codomain is \(\R\text{.}\)
