Following is a statement of a theorem which can be proven using calculus or precalculus mathematics. For this theorem, \(a\text{,}\) \(b\text{,}\) and \(c\) are real numbers.

Theorem

If \(f\) is a quadratic function of the form \(f\left( x \right) = ax^2 + bx + c\) and \(a \lt 0\text{,}\) then the function \(f\) has a maximum value when \(x = \dfrac{{ - b}}{{2a}}\text{.}\)
