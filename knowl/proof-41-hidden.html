Let \(f\x A \to B\) and \(g\x B \to C\) be bijections. Then \(f^{ - 1} \x B \to A\) and \(g^{ - 1} \x C \to B\text{.}\) Hence, \(f^{ - 1} \circ g^{ - 1} \x C \to A\text{.}\) Also, by Theorem 6.4.8, \(g \circ f\x A \to C\) is a bijection, and hence \(( {g \circ f} )^{ - 1} \x C \to A\text{.}\) We will now prove that for each \(z \in C\text{,}\) \(( {g \circ f} )^{ - 1} ( z ) = ( {f^{ - 1} \circ g^{ - 1} } )( z )\text{.}\)

Let \(z \in C\text{.}\) Since the function \(g\) is a surjection, there exists a \(y \in B\) such that

\begin{equation} g( y ) = z\text{.}\tag{6.7} \end{equation}

Also, since \(f\) is a surjection, there exists an \(x \in A\) such that

\begin{equation} f( x ) = y\text{.}\tag{6.8} \end{equation}

Now these two equations can be written in terms of the respective inverse functions as

\begin{align} g^{ - 1} ( z ) \amp = y; \text{ and }\tag{6.9}\\ f^{ - 1} ( y ) \amp = x\text{.}\tag{6.10} \end{align}

Using equation (6.9) and equation (6.10), we see that

\begin{align} \left( {f^{ - 1} \circ g^{ - 1} } \right)( z ) \amp = f^{ - 1} \left( {g^{ - 1} ( z )} \right)\notag\\ \amp = f^{ - 1} ( y )\notag\\ \amp = x\text{.}\tag{6.11} \end{align}

Using equation (6.7) and equation (6.8) again, we see that \(( {g \circ f} )( x ) = z\text{.}\) However, in terms of the inverse function, this means that

\begin{equation} ( {g \circ f} )^{ - 1} ( z ) = x\text{.}\tag{6.12} \end{equation}

Comparing equation (6.11) and equation (6.12), we have shown that for all \(z \in C\text{,}\)

\(( {g \circ f} )^{ - 1} ( z ) = ( {f^{ - 1} \circ g^{ - 1} } )( z )\text{.}\) This proves that \(( {g \circ f} )^{ - 1} = f^{ - 1} \circ g^{ - 1}\text{.}\)
