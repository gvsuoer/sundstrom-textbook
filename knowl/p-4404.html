Let \(f\x \mathbb{R} \to \mathbb{R}\) be defined by \(f( x ) = x^2\text{,}\) for each \(x \in \mathbb{R}\text{.}\) The following results are based on the examples in Beginning Activity 1 and Beginning Activity 2.

  • Let \(A = \left\{ 1, 2, 3, -1 \right\}\text{.}\) Then \(f ( A ) = \left\{ 1, 4, 9 \right\}\text{.}\)

  • Let \(B = \left\{ 1, 9, 15, -1 \right\}\text{.}\) Then \(f^{-1} ( B ) = \left\{ -\sqrt{15}, -3, -1, 1, 3, \sqrt{15} \right\}\text{.}\)
