Let \(n\) be a natural number and let \(a, b, c\text{,}\) and \(d\) be integers. Then, in \(\mathbb{Z}_n\text{,}\)

  1. If \([ a ] = [ b ]\) and \([ c ] = [ d ]\text{,}\) then \([ {a + c} ] = [ {b + d} ]\text{.}\)

  2. If \([ a ] = [ b ]\) and \([ c ] = [ d ]\text{,}\) then \([ {a \cdot c} ] = [ {b \cdot d} ]\text{.}\)

  3. If \([ a ] = [ b ]\) and \(m \in \mathbb{N}\text{,}\) then \([ a ^m] = [ b ^m]\text{.}\)
