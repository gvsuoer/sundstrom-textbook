Example 1.19. Set Builder Notation.

Let \(B = \left\{ \ldots, -11, -7, -3, 1, 5, 9, 13, \ldots \right\}\text{.}\) The key to writing this set using set builder notation is to recognize the pattern involved. We see that once we have an integer in \(B\text{,}\) we can obtain another integer in \(B\) by adding 4. This suggests that the predicate we will use will involve multiplying by 4.

Since it is usually easier to work with positive numbers, we notice that \(1 \in B\) and \(5 \in B\text{.}\) Notice that

\begin{equation*} 1 = 4 \cdot 0 + \text{ and } 5 = 4 \cdot 1 + 1\text{.} \end{equation*}

This suggests that we might try \(\left\{ 4n + 1 \mid n \in \Z \right\}\text{.}\) In fact, by trying other integers for \(n\text{,}\) we can see that

\begin{equation*} B = \left\{ \ldots, -11, -7, -3, 1, 5, 9, 13, \ldots \right\} = \left\{ 4n + 1 \mid n \in \Z \right\}\text{.} \end{equation*}
