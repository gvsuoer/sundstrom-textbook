Define the relation \(\sim\) on \(\Z\) as follows: For \(a, b \in \Z\text{,}\) \(a \sim b\) if and only if \(2a + 3b \equiv 0 \pmod 5\text{.}\) The relation \(\sim\) is an equivalence relation on \(\Z\text{.}\) (See Exercise 13 in Section 7.2). Determine all the distinct equivalence classes for this equivalence relation.
